>本仓库是「风绘的小窝」个人随记博客的存储库，基于[Fuwari](https://github.com/saicaca/fuwari)静态博客模版搭建，使用了Edgeone Pages提供部署支持！

### 小窝的简单介绍

- 博客地址：https://blog.fhowo.top
- 内容方向：聚焦原创技术教程、博客搭建经验分享及生活随笔。​
- 核心功能：继承 Fuwari 模板的所有特性，并针对个人需求进行了定制化调整，包括主题色优化、导航栏*友人帐* 。
